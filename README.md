Features:
1. Authentication
2. Responsive Design
3. Property Card
4. Interactive Slides

Packages Used
  React.js
  Tailwind CSS
  Firebase
  Swiper.js
  AOS (Animate on Scroll):
  React Icons



Installation
To get started with the project locally, follow these steps:

Clone the repository:
  
  git clone https://github.com/yourusername/next-door-realty.git
  
  cd next-door-realty

Install dependencies:
  
  npm install
